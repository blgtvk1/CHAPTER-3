# 🚀 LexiaBuddy: Erişilebilir Eğitimde Yapay Zeka Dokunuşu

**LexiaBuddy**, disleksi tanısı almış çocukların eğitim materyallerine erişimini kolaylaştırmak için geliştirilmiş, empati odaklı bir yapay zeka çözümüdür.

---

## 📝 Proje Hakkında
Eğitimde fırsat eşitliği, yalnızca kaynaklara ulaşmakla değil, o kaynakları anlamlandırabilmekle başlar. Boğaziçi Üniversitesi’nde Psikolojik Danışmanlık ve Rehberlik eğitimi alırken, disleksi tanısı almış çocukların akademik materyallerle kurduğu zorlu ilişkiyi gözlemleme fırsatım oldu. Bu gözlemi, teknolojiye olan tutkumla birleştirerek **LexiaBuddy** projesini hayata geçirdim.

**LexiaBuddy**, karmaşık akademik metinleri veya taranmış PDF belgelerini yapay zeka (LLM) desteğiyle disleksi dostu formatlara dönüştüren bir web uygulamasıdır. Projenin kalbinde, bilişsel yükü azaltmak için tasarlanmış özel bir mimari yer alıyor.

## ✨ Temel Özellikler
* **Akıllı Sadeleştirme:** Uzun ve karmaşık cümleleri, anlam kaybı olmadan kısa ve öz ifadelere dönüştürür.
* **Bilişsel Tasarım:** Metinleri, disleksi bireyler için özel olarak geliştirilen **Lexend** yazı tipiyle yeniden yapılandırır.
* **Görsel Düzenleme:** Paragraf aralıklarını ve liste yapılarını okuma kolaylığı sağlayacak şekilde optimize eder.
* **Kapsayıcı Teknoloji:** JavaScript ve Gemini API entegrasyonu ile metin bariyerlerini ortadan kaldırır.

## 🛠️ Kullanılan Teknolojiler
* **Frontend:** HTML5, CSS3 (Modern ve duyarlı arayüz)
* **Yapay Zeka:** Groq / Gemini API (Doğal Dil İşleme)
* **PDF İşleme:** PDF.js (Metin çıkarma ve yapılandırma)
